LENGUAJE DE MARCAS

En diseño web, un lenguaje de marcas se refiere a la utilización de HTML y lenguajes similares para estructurar
y presentar contenido en la web. HTML es el núcleo, utilizando etiquetas y elementos para definir la jerarquía
y el formato del contenido, como encabezados, párrafos, imágenes y enlaces.

Las etiquetas HTML contienen instrucciones para definir un elemento HTML específico e indican cómo mostrarlo en la página.

Cada etiqueta HTML comienza con un símbolo de menor que "<" y termina con un símbolo de mayor que ">" , y preceden
y siguen al contenido del elemento

Por ejemplo:

p: Se utiliza para definir un parrafo
h: Son del 1 al 6 y sirven para los encabezados
a: Crea un enlace a otra pagina o recurso
img: Inserta una imagen en la pagina
br: crea un salto de linea

ELENTOS DE CONTROL

Los elementos de control en HTML son componentes interactivos que permiten a los usuarios ingresar
y manipular datos dentro de una página web, tambien comienza con un símbolo de menor que "<" y termina
con un símbolo de mayor que ">"

Por ejemplo:

button: crea un boton,se puede usar como enlace o para desplegar informacion
textarea: Permite ingresar texto en múltiples líneas, ideal para comentarios o textos largos.
label: Asocia una etiqueta a un elemento de entrada, mejorando la accesibilidad y la usabilidad.
input type="color": Muestra un selector de color para que el usuario elija un color específico.
input type="tel": Permite ingresar números de teléfono, ideal para formularios de contacto.
input type="password": Crea un campo para ingresar contraseñas, enmascarando el texto con
puntos o asteriscos.
